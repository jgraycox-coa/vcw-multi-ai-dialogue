# Prompt Library

This directory contains the complete prompt library for the VCW Multi-AI Dialogue experiment.

## Files

- `background_document.md` - The 4,963-word theoretical foundations document provided to all participants
- `proposer_prompts.md` - Turn-specific prompts for the Proposer role (6 variants)
- `responder_prompts.md` - Turn-specific prompts for the Responder role (6 variants)
- `monitor_prompt.md` - Assessment prompt for the Monitor role
- `translator_prompt.md` - Summary prompt for the Translator role

## Usage

These prompts are designed to be used with the `vcw_integration_v4.py` orchestration framework.
Modify the background document to test different alignment frameworks.
